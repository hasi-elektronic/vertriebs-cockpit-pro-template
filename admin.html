Backup & Daten-Export

Lade ein vollstaendiges JSON-Backup aller Daten herunter. Enthaelt: Benutzer, Pipeline-Status, Notizen, Firminfo-Overrides, Konkurrenz-Klassifikationen, Kontakte, Aktivitaet, Score-Overrides, News, Audit-Log.

Hinweis: Backup ist eine ZIP-loose JSON-Datei. Bewahre sie an einem sicheren Ort auf (USB-Stick, lokale Festplatte). Im Notfall (KV-Verlust, Account-Compromise) kann der Stand wiederhergestellt werden.

Sprint 10 Sicherheit aktiv:
  • Brute-Force Schutz: 5 Login-Versuche pro IP / 15min
  • CSRF: SameSite=Strict + Origin-Check auf allen API-Endpunkten
  • Session: 7 Tage (vorher 30) + HttpOnly + Secure
  • Input-Limits: 200 Zeichen Felder / 5000 Zeichen Notizen
  • HTTP-Cache: leads-geo.json 10min, andere Pages no-cache
  • HSTS, CSP, X-Frame-Options aktiv
